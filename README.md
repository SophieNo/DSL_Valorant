# Projet-Scala

Bienvenue dans le **Projet-Scala**. Ce projet consiste à développer un DSL (Domain-Specific Language) pour fournir une interface simple et intuitive permettant de générer des stratégies, des conseils d'achat, et des suggestions de composition pour le jeu Valorant. Toutes les informations dur le projet sont disponible sur le Wiki du dépôt GitHub.
